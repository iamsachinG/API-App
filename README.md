📌 Random Cat Fact Generator

A simple web app that displays random cat facts using an external API. Each time you refresh or trigger the function, a new fact is shown on the screen.

🚀 Features
Fetches random cat facts from an API
Dynamic content update using JavaScript
Clean and minimal UI
Beginner-friendly project

🛠️ Tech Stack
HTML
CSS
JavaScript (Fetch API)

📂 Project Structure
API-App/
│── index.html   # Main UI
│── script.js    # Fetch API + logic

⚙️ How It Works
JavaScript sends a request to a Cat Facts API
API responds with a random fact
Fact is displayed on the UI dynamically
